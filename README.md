# CodeAlpha Tasks

This repository contains Python mini projects completed as part of CodeAlpha tasks.

## Task 1: Chatbot
A simple rule-based chatbot that responds to basic user inputs like "hello", "how are you", and "bye". It uses basic Python concepts such as if-elif conditions, loops, and input/output.

## Task 2: Hangman Game
A text-based Hangman game where the user guesses a word one letter at a time. The game allows a maximum of 6 incorrect guesses and displays the progress after each attempt.

## Technologies Used
- Python

## How to Run
Download or clone this repository and run the following commands in terminal:
python chatbot.py  
python hangman_game.py
